# 📊 AI ML Prediction Web App (Streamlit)

## 🚀 Project Overview
This is an end-to-end Machine Learning web application built using **Python, Scikit-learn, and Streamlit**.  
The model predicts outcomes based on user input data and provides real-time results through an interactive UI.

---

## 🎯 Objective
To build a machine learning system that:
- Processes input data
- Applies trained ML model
- Outputs prediction in real-time via web interface

---

## 🧠 Machine Learning Pipeline
1. Data Collection
2. Data Preprocessing (handling missing values, scaling, encoding)
3. Model Training (Scikit-learn)
4. Model Evaluation
5. Model Serialization (Joblib / Pickle)
6. Deployment using Streamlit

---

## 📊 Model Performance
- Accuracy: **~81.33%**
- Evaluation: Train/Test split validation
- Metrics used: Accuracy Score

---

## 🛠️ Tech Stack
- Python 🐍
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib
